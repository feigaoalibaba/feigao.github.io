# 毕业证
国家承认学历，学信网（www.chsi.com.cn）永久可查，教育部电子注册。正规流程毕业发证。

# 报名资料
身份证电子版、一寸、二寸蓝底照片电子版各一份、最高学历证明复印件。

# 费用支付方式
成人高考首次报名预付总费用50%，学信网查询到学籍注册信息后一次性补齐余额。自考按考取科目收取。

# 考试地点
张家口或廊坊
